# Scout Plugin SDKs

SDKs for building Scout plugins in multiple languages.

| Language | Location | Package |
|----------|----------|---------|
| Go | `pkg/scout/plugin/sdk/` | `github.com/inovacc/scout/pkg/scout/plugin/sdk` |
| TypeScript | `sdk/typescript/` | `@inovacc/scout-plugin-sdk` |
| Python | `sdk/python/` | `scout-plugin-sdk` |

All SDKs implement the same JSON-RPC 2.0 protocol over stdin/stdout.

See [Plugin System documentation](../CLAUDE.md) for the full protocol specification.
